### Features

* **Live Timer & Mistake Tracker:** Keep track of your speed and precision as you complete the daily puzzle.
* ** Cosmic Navy Aesthetic:** Minimalist slate-navy interface inspired by vintage puzzle prints and clean dark-mode design.
* ** Midnight UTC Automation:** Powered by GitHub Actions, refreshing automatically every day at 00:00 UTC.
* ** Dynamic Score Calculation:** Computes a score based on time completed and mistake penalties.
